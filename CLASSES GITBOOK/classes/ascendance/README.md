# Ascendance

Most classes stop gaining features at level 11, but every class universally has access to ascendance, which is granted at level 12. After which the content train stops. Ascendances are meant to cater to a fantasy where after a character's journey has gone on long enough, they not only acquire power, but also often prestige. Ascendances can often be summed up in a single title like "immortal" or "archmage" or "demigod."
