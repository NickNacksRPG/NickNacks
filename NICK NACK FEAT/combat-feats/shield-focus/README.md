# Shield Focus

<mark style="color:green;">Requirements</mark> -&#x20;

You can use a shield feature as an instant action 1/round.

Gain a new shield action, usable as an immediate action.\
<mark style="color:green;">Shield Crash</mark> - When a creature misses you or grazes you with a melee attack, you can knock them down (fort negates).
