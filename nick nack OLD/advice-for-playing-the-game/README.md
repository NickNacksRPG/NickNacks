# Advice for Playing the Game

Playing tabletop can be a daunting experience. With so many rules and mechanics that you are unfamiliar with, it may discourage you from playing. This section is to give you advice on how to make the process feel a bit less daunting.

If you don't really care about what I have to say about playing the game, then skip ahead to the rules. This section is mostly intended for prospective players and game masters...Or dungeon masters, or keepers or whatever funny name you call it.
