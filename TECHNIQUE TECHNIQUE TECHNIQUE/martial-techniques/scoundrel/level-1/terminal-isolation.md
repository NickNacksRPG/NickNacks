# Terminal Isolation

## <mark style="color:orange;">Terminal Isolation \[upcast]</mark>

## <mark style="color:orange;">Scoundrel 1</mark>

## <mark style="color:orange;">Cast Time</mark> - 1 Special Attack

<table><thead><tr><th width="161">Effect</th><th></th></tr></thead><tbody><tr><td>Target</td><td>1 "isolated" creature</td></tr></tbody></table>

{% hint style="info" %}
This technique can only be used against an isolated creature. A creature is considered isolated when...

* It does not have an ally within 15 ft.
* It is not flanking with an ally
{% endhint %}

Strike at an isolated creature with a melee attack, adding d8 (4) dmg.

<mark style="color:green;">Upcast</mark> - For each additional mana spent, deal an additional d8 (4) damage.
