# Marksman

The techniques surrounding using ranged weapons. Centers around a high ground mechanic.&#x20;

{% hint style="success" %}
**High Ground** requires you to have a 10ft height advantage over your opponent. This height advantage cannot be achieved with flight or jumping. You must be standing on solid ground. AKA, its not just about height, but rather being in an advantageous position.
{% endhint %}

Many marksman special attacks require using ranged weapons. As such, they use the weapon you wield's range.
