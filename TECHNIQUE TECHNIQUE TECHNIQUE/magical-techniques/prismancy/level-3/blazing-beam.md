# Blazing Beam

## <mark style="color:orange;">Blazing Beam \[fire] \[upcast]</mark>

## <mark style="color:orange;">Prismancy 3</mark>

## <mark style="color:orange;">Cast Time</mark> - 1 Standard Action

<table><thead><tr><th width="161">Effect</th><th></th></tr></thead><tbody><tr><td>Range</td><td>30 ft</td></tr><tr><td>Target</td><td>1 creature</td></tr></tbody></table>

Fire a beam that magnetizes to burning creatures and feeds the flames. The target takes 4d8+CL (18+CL) fire damage. If they are ablaze, the DC to save against the condition is increased by 4, and the penalty to AC is increased to -4. Only standard actions grant a new save against being ablaze. If an effect would normally automatically remove ablaze, it instead only grants a save without the -4 penalty.

<mark style="color:green;">Upcast</mark> - For each additional mana spent, the spell deals an additional d8 dmg.
