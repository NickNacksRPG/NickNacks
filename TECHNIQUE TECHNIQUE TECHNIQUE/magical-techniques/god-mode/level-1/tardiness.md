# Tardiness

## <mark style="color:orange;">Tardiness</mark>

## <mark style="color:orange;">God Mode 1</mark>

## <mark style="color:orange;">Cast Time</mark> - 1 Standard Action

<table><thead><tr><th width="161">Effect</th><th></th></tr></thead><tbody><tr><td>Range</td><td>60 ft</td></tr><tr><td>Target</td><td>1 creature</td></tr><tr><td>Save</td><td>Will</td></tr></tbody></table>

Delay a creature so that way their spot in initiative is moved to be dead last (will negates). If the creature has already acted in this round, it does not get an additional turn.
