# Blazing Bulwark

## <mark style="color:orange;">Blazing Bulwark \[fire]</mark>

## <mark style="color:orange;">Pyromancy 1</mark>

## <mark style="color:orange;">Cast Time</mark> - 1 Immediate Action

<table><thead><tr><th width="161">Effect</th><th></th></tr></thead><tbody><tr><td>Target</td><td>1 creature</td></tr><tr><td>Save</td><td>Reflex</td></tr></tbody></table>

{% hint style="info" %}
Special: Must be wielding a shield to cast\
If you have [shield focus](https://app.gitbook.com/s/vxnMGGHnEtmcEQDFxcK6/combat-feats/shield-focus), you can cast this spell as an instant action.
{% endhint %}

As part of casting the spell, take the block action. If the foe attacked with a melee weapon, they are set ablaze (reflex negates).
