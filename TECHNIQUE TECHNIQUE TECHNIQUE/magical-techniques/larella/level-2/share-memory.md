# Share Memory

## <mark style="color:orange;">Share Memory \[concentration] \[mind-affecting]</mark>

## <mark style="color:orange;">Larella 2</mark>

## <mark style="color:orange;">Cast Time</mark> - 1 Standard Action

<table><thead><tr><th width="161">Effect</th><th></th></tr></thead><tbody><tr><td>Range</td><td>Reach</td></tr><tr><td>Target</td><td>1 creature</td></tr><tr><td>Save</td><td>Will</td></tr></tbody></table>

With a touch, you can tap into your targets memories (will negates) or your own, decided when you initiate mental contact. In any case, both parties see what is being viewed. The memory is limited to one minute duration. If gazing into another creatures memories, you must specify the type of memory or specific memory you want to look at.
